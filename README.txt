_{word_repeat}_

#####_{Takes a string and tells how many of each word there is} {8/07/2015}

####By _**{Travis Crampton}

## Description

_{Application first takes a string from a user in the first input area, for example "Travis Loves Loves to program", the second input area provided will as the user to input a single word from the string. ex. "loves" after submitting both forms the program will take the string, convert it to an array and then assemble a hash with the words in the array counting how many times that the word actually occurs. The end result spits out the number that the input is associated with. It is also programmed to respond with "That word isn't in the text" if the search word of the input is not included in the first string sentence.}

##Setup

install bundler
bundle install Gemfile
run sinatra server
